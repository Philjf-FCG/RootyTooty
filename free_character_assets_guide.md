# Free Character Assets Guide

This project now supports optional free cosmetic assets for character styling.

## Folder Setup

Put downloaded FBX files here:
- `Assets/`
- `Assets/FreeWestern/`

Then run Unreal Python script:
- `import_hats_from_root.py`

Imported destination paths:
- `/Game/Assets`
- `/Game/Assets/FreeWestern`

## Recommended Asset Names

If you name imported static meshes like these, runtime auto-detection will pick them up:
- `SM_SheriffHat`
- `SM_SheriffHat_Crown`
- `SM_SheriffHat_Brim`
- `SM_BanditHat`
- `SM_BanditHat_Crown`
- `SM_BanditHat_Brim`
- `SM_Bandana`

## Free Sources (verify license before use)

- Kenney (CC0): https://kenney.nl/assets
- Quaternius (free game-ready packs): https://quaternius.com
- OpenGameArt (mixed licenses): https://opengameart.org

Use only assets with redistribution/commercial rights that match your project needs.
